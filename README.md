# Customer Segmentation Using RFM Analysis

## Project Overview
This project performs customer segmentation using the **RFM model (Recency, Frequency, Monetary)**. RFM analysis helps businesses identify their most valuable customers based on purchasing behavior.

The goal is to group customers into segments that help businesses improve marketing strategies and customer retention.

---

## Objectives
- Analyze customer purchase behavior
- Calculate Recency, Frequency, and Monetary values
- Segment customers based on purchasing patterns
- Identify high-value and inactive customers

---

## Dataset
The dataset used is the **Online Retail dataset**, which contains transaction records from an online store.

Important columns include:
- InvoiceNo
- StockCode
- Description
- Quantity
- InvoiceDate
- UnitPrice
- CustomerID
- Country

---

## Tools and Technologies
- Python
- Pandas
- NumPy
- Matplotlib

---

## Project Workflow
1. Load and explore the dataset
2. Clean the dataset and remove missing values
3. Calculate Recency (last purchase date)
4. Calculate Frequency (number of purchases)
5. Calculate Monetary value (total spending)
6. Assign RFM scores
7. Segment customers into groups

---

## Customer Segments Identified
- Loyal Customers
- High-Value Customers
- New Customers
- At-Risk Customers
- Inactive Customers

---

## Key Insights
- A small group of customers contributes to a large percentage of total revenue.
- Loyal customers make frequent purchases with high monetary value.
- Some customers have not made purchases recently and require re-engagement strategies.

---

## Conclusion
RFM analysis helps businesses better understand their customers and develop targeted marketing strategies to increase retention and revenue.
